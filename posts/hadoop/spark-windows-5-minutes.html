Spark on Windows in Under 5 Minutes

Spark can run over a variety of cluster manager, including Hadoop YARN, Apache Mesos, and Spark itself called Standalone Scheduler. In this recipe we are going to prepare Windows for running Spark, create and execute a Spark workflow. Installing Spark under Linux or MacOS is very similar.

To get Spark up and running, you need to complete following tasks:

  1. Make sure Java Development Kit (JDK) is Installed
  2. Download and Install Apache Spark
  3. Configure Windows Environment

Make Sure JDK is Installed

If you are not sure, you have JDK installed, better is to install the JDK.

  1. Open a browser window and navigate to your favorite search engine.
  2. Search for JDK SE Download and follow the link to the Oracle's JDK download page.
  3. Click the button to JDK Download.
  4. Accept the License Agreement and click the link to the package for you operating system. Most likely this will be the Windows package.
  5. Once the dowload is complete. Run the installer and follow the installation steps.

Install Apache Spark

  1. Navigate your browser to the Apache Spark home.
  2. Configure your download:
    • Choose a Spark release - Select the latest version
    • Choose a package type - Select Pre-built for Hadoop 2.6 and later
    • Choose a dowload type - Direct Download
  3. Click the package download link and save the package to your disk.
  4. Use your favorite archiver (my favorite is 7-zip) to extract spark. I will assume you extracted the archive into your root directory (C:\)
  5. Rename the spark directory to C:\spark
