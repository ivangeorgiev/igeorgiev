Web Page Mockups with JavaScript Generated Placeholders

Using placeholder images is a very useful technique for rapid web page prototyping. Placeholder images are dummy, wireframe images that replace the actual image on the web page.

Here are examples of what we will create in this tutorial.

There are different approaches to the placeholder images.

  • You can create placeholder images for each necessary size.
  • You can use an online service like http://placehold.it
  • Use javascript to generate placeholder images.

Creating prototype images for each size is boring and time consuming task. The online service provides flexible way of generating placeholder images with specified size. Disadvantages of this approach include the need for persistent online connection and fixed visual appearance.

It is possible to generate placeholder images on the client side. Here you have also many possible options. Here I will show you how to create placeholders using JavaScript and scalable vector graphics (SVG). This approach produce flexible and fully customizable images. You can control the appearance of the placeholders via CSS or if you want to create diffferent placeholders, you can modify the JavaScript.

This is an intermediate tutorial. I will not go into details explaining every detail. I will assume that you already have working knowledge on HTML, CSS, JavaScript and jQuery. You should also have basic SVG understanding.

Building a Prototype

Let's start by manually creating a prototype that produces a 350-by-150 pixels placeholder image.


<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.placeholder-box {
  fill: lightsteelblue;
  stroke: steelblue;
  stroke-width: 1px;
}

.placeholder-line {
  stroke: steelblue;
  stroke-width: 1px;
}

.placeholder-label rect {
  fill: lightsteelblue;
  stroke: none;
}

.placeholder-label text {
  font: 14px sans-serif;
  fill: steelblue;
}
</style>
</head>

<body>
    <svg width="350" height="150">
      <g>
        <rect x="0" width="350" y="0" height="150" class="placeholder-box"></rect>
        <line x1="0" y1="0" x2="350" y2="150" class="placeholder-line"></line>
        <line x1="350" y1="0" x2="0" y2="150" class="placeholder-line"></line>
        <g class="placeholder-label" transform="translate(125,70)">
          <rect y="-2" height="16" width="100"></rect>
          <text x="50" dy="10" text-anchor="middle">350x150</text>
        </g>
      </g>
    </svg>
</body>
</html>

Here we created a SVG element that includes a 350x150 pixels rectange and two diagonal lines. In the middle of the rectange we placed a label. To enable CSS styling, we added class attributes to the SVG elements:

  • placeholder-box - the enclosing placeholder rectangle. You can customize stroke and fill. Interesting efects can be achieved with rounded corners.
  • placeholder-line - the two diagonal lines. You can customize stroke. Typically the stroke for the lines is the same as the stroke for the placeholder box rectangle.
  • placeholder-label - this class is added to the group that holds the label's text element and background rectangle.

Creating Placeholder Generation JavaScript

First we modify our HTML code so that it contains only a div with attribute data-placeholder="350x150". We are going to use jQuery to select all the div elements with data-placeholder attribute and insert corresponding svg elements. At the end of the HTML, just before the closing body are inserted script elements for jQuery and our placeholder.js (will show it to you in a minute). 


<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title></title>
  <link rel="stylesheet" href="style.css">
</head>

<body>
  <div data-placeholder="350x150"></div>
  <script src="js/jquery-1.11.3.min.js"></script>
  <script src="placeholder.js"></script>
</body>
</html>

Here I have also moved the CSS code into external stylesheet style.css.

[style.css]


.placeholder-box {
  fill: lightsteelblue;
  stroke: steelblue;
  stroke-width: 1px;
}

.placeholder-line {
  stroke: steelblue;
  stroke-width: 1px;
}

.placeholder-label rect {
  fill: lightsteelblue;
  stroke: none;
}

.placeholder-label text {
  font: 14px sans-serif;
  fill: steelblue;
}

And here is the JavaScript code. The placeholder JavaScript code is stored in a placeholder.js file.

[placeholder.js]


$(function() {
  $('[data-placeholder]').each(function(_,ph) {
    var el = $(ph),
        data = el.data('placeholder'),
        dim = (data) ? data.split('x') 
                     : [el.width(), el.height()],            
        w = dim.shift() || el.width(),
        h = dim.shift() || el.height(),
        svg;

    svg = $('<svg></svg>')
      .attr('width', w)
      .attr('height', h)
      .appendTo(ph)
        .append(
          $('<rect x="1" y="1"></rect>')
            .attr('width', w-2)
            .attr('height', h-2)
            .addClass('placeholder-box')
        ) // rect
        .append(
          $('<line x1="5", y1="5"></line>')
            .attr('x2', w-5)
            .attr('y2', h-5)
            .addClass('placeholder-line')
        ) // line1
        .append(
          $('<line x2="5", y1="5" />')
            .attr('x1', w-5)
            .attr('y2', h-5)
            .addClass('placeholder-line')
        ) // line2
        .append(
          $('<g class="placeholder-label" />')
            .append(
              $('<rect width="100" height="18" />')
                .attr('x', w/2 - 50)
                .attr('y', h/2 - 9)
            )
            .append(
              $('<text text-anchor="middle" />')
                .attr('x', w/2)
                .attr('y', h/2 + 4)
                .append(w+'x'+h)
            )
        ) // label
    ;
    el.width(w);
    el.height(h);

    el.html(el.html());
  });
})

Now if you open the page into a browser, you should see a generated placeholder image.

Styling Examples

Here are some examples of different placeholder styling.

Placeholder Images Without Lines

If you set stroke: none for the .placeholder-line, you will remove the placeholder lines.


.placeholder-box {
  fill: lightcyan;
  stroke: cyan;
  stroke-width: 1px;
}

.placeholder-line {
  stroke: none;
}

.placeholder-label rect {
  fill: lightcyan;
  stroke: none;
}

.placeholder-label text {
  font: 14px sans-serif;
  fill: steelblue;
}

Similarly, set fill: none for .placeholder-label text to remove the text label.

Placeholder Images with Rounder Corners

Add rx and ry CSS rules for .placeholder-box to create rounder corner placeholder image.


.placeholder-box {
  fill: lightsteelblue;
  stroke: steelblue;
  stroke-width: 1px;
  rx: 10px;
  ry: 10px;
}

Excercises

  1. Elliptical/Circular image. Try using percentage values for rx and ry CSS rules to achieve circular/elliptiical image.
  2. Custom label text. Introduce data-text attribute. 
    <div data-placeholder="150x150" data-text="avatar picture"></div>
  3. Extend the JavaScript so that the label text can be rendered vertically. The text direction is specified using a data-text-dir attribute:
    <div data-placeholder="150x150" data-text-dir="vertical"></div>
