Client Side Code Syntax Highlighting with JavaScript and prettify.js

Prettify script can be used to highlight code on web pages. It is a client side JavaScript and works entirely in the browser.

You can download the Prettify library from github or link it on github:

<script src="https://cdn.rawgit.com/google/code-prettify/master/loader/run_prettify.js"></script>

Highlight Code

To highglight source code, enclose your code in `code` tag and add `prettyprint` class to the `pre` tag. Here is an example of a Python that converts temperature value from Celsius to Fahrenheit:

<pre  class="prettyprint">
def celsius_from_fahrenheit(temp):
    return (temp - 32)*5.0 / 9.0
</pre>

The browser will render this as follows:

def celsius_from_fahrenheit(temp):
    return (temp - 32)*5.0 / 9.0

As you can see the Prettify script is very easy to use. Specifying the language is not required, but if you like to hint the script, you can add a language hint:

<pre  class="prettyprint lang-py">
def celsius_from_fahrenheit(temp):
    return (temp - 32)*5.0 / 9.0
</pre>

Highlight Skin

You can go beyond the default skin and select different skin. Add `skin=skinname` url parameter to the javascript src attribute.

<script src="https://cdn.rawgit.com/google/code-prettify/master/loader/run_prettify.js?skin=desert"></script>

Line Numbers

By default no line numbers are shown with the highlighted code. To enable line numbers, add `linenums` class to the enclosing `pre` tag. If you want to start numbering from a certain line number, for example 123, specify the class as `ilnenums:120'.

<pre  class="prettyprint linenums:120">
def celsius_from_fahrenheit(temp):
    """
    Convert temperature value from Fahrenheit to Celsius.
    """
    return (temp - 32)*5.0 / 9.0
    
def fahrenheit_from_celsius(temp_fahrenheit):
    """
    Convert temperature value from Celsius to Fahrenheit.
    """
    return (temp_fahrenheit*9 + 160) /
    
</pre>

And the result.

def celsius_from_fahrenheit(temp):
    """
    Convert temperature value from Fahrenheit to Celsius.
    """
    return (temp - 32) * 5.0 / 9.0
    
def fahrenheit_from_celsius(temp_fahrenheit):
    """
    Convert temperature value from Celsius to Fahrenheit.
    """
    return (temp_fahrenheit * 9 + 160) / 5.0

Yes, you are right. Line numbers are shown only on each fifth line. This is a feature. To turn on the line numbering for all the lines, you need to modify the document stylesheet. Add following:

<style>
li.L0, li.L1, li.L2, li.L3,
li.L5, li.L6, li.L7, li.L8
   { list-style-type: decimal !important }
</style>
